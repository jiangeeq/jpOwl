### 模块说明

1. **jpowl-core**: 核心功能模块
    - 提供基础监控能力
    - 实现数据采集和处理
    - 支持多种输出方式
    - 包含核心数据模型

2. **jpowl-spring**: Spring框架集成
    - 提供Spring集成支持
    - 实现Actuator端点
    - 配置自动装配
    - 提供模板类

3. **jpowl-spring-boot-starter**: Spring Boot 启动器
    - 自动配置
    - 条件装配
    - 外部化配置

4. **扩展模块**:
    - jpowl-extension-alert: 告警通知
    - jpowl-extension-logging: 日志扩展
    - jpowl-extension-output: 输出扩展

5. **示例模块**:
    - jpowl-spring-samples: 使用示例
    - 包含常见场景演示
